Як-52 — советский спортивно-тренировочный самолёт.

Представляет из себя двухместный цельнометаллический моноплан с низкорасположенным свободнонесущим крылом. Являлся одним из основных учебно-тренировочных самолётов СССР для первоначальной подготовки лётного состава.
